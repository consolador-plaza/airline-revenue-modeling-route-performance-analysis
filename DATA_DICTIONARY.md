# Purpose

This document describes the structure of the project's analytical dataset.

The dataset

```
dbo.route_performance_analytics
```

serves as the single source of truth for SQL validation, exploratory analysis, Power BI reporting, and project documentation.

---

# Analytical Grain

Each record represents:

```
Year

Quarter

Origin Airport

Destination Airport
```

or

> One airport pair during one calendar quarter.

# Derived Metrics

## Estimated Passenger Revenue

**Formula**

```
Passengers × Weighted Average Fare
```

Purpose

Provides an estimate of commercial passenger revenue at the airport-pair level.

---

## Weighted Average Fare

**Formula**

```
SUM(Fare × Passengers)

÷

SUM(Passengers)
```

Purpose

Calculates passenger-weighted fares to reduce statistical bias.

---

## Load Factor

**Formula**

```
Passengers

÷

Seats
```

Purpose

Measures seat utilization.

---

## Revenue per Passenger

**Formula**

```
Estimated Revenue

÷

Passengers
```

Purpose

Measures average estimated commercial value generated by each passenger.
